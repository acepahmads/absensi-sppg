SCADA & Automation System Integration with Optional Direct PLC/RTU Access

Automation System operates as advisory and analytical layer, with controlled read-only access to PLC/RTU via Modbus RTU/TCP.

⚠️

Direct PLC/RTU connection is optional and read-only

🚫

No direct control or write access from Automation System

✓

SCADA remains the primary control and monitoring system
